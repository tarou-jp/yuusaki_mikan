# 言語モデル学習用対話文収集アプリ(未完成)

##　概要
本プロジェクトでは、ユーザーに特定のキャラになりきって会話をしてもらうアプリ開発を行った。ユーザーがキャラになりきって行った会話を用いて言語モデルの学習を行う予定であったが、
使用できるキャラクターが見つからなかったため、完成、公開には至らなかった。
